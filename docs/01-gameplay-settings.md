# 01 — Gameplay & Controller Settings (PS5)

Settings win more games than tactics. Get these right once and never think about
them again.

## Gameplay preset: Competitive vs Authentic

FC 26 ships two gameplay feels:

- **Competitive** — faster, more responsive, more "arcadey." This is what online
  Rivals / Champions / ranked play uses. **If you play online, practice on
  Competitive** so muscle memory matches your ranked games.
- **Authentic** — slower, more sim-like, heavier momentum and fatigue. Better for
  immersion and Career Mode. Use it offline if you enjoy a realistic pace.

> Rule of thumb: train in the same gameplay mode you compete in.

## Controller settings (the important ones)

| Setting | Recommended | Why |
|---|---|---|
| **Auto Switching** | Manual or Air Balls | Forces you to learn manual switching (R-stick). The defensive skill gap lives here. |
| **Auto Switch Move Assistance** | Off | Stops the game stealing control of your defender mid-press. |
| **Right Stick Switching** | On / Precise | Flick the R-stick toward the player you want — far more reliable than tapping L1/LB. |
| **Jockey** | Manual | You want full control of contain. |
| **Pass Assistance** | Semi (drop from Assisted once comfortable) | Semi rewards aim and unlocks passes assisted refuses to make. |
| **Through Pass Assistance** | Semi | Threaded through balls become a weapon. |
| **Shot Assistance** | Assisted or Semi | Assisted is fine for most; Semi gives finer placement at a consistency cost. |
| **Save Assistance** | Manual (advanced) / Assisted (default) | Manual GK saves are a high-risk, high-reward edge in 1v1s. |
| **Auto Flair Passes** | Off | No accidental backheels gifting possession. |

## Camera

- **Camera:** Tele Broadcast or Co-op.
- **Height:** 20 · **Zoom:** 0–5.
- Goal is to **see the whole half** so you can read runs and pick line-breaking
  passes. A zoomed-in camera hides space and gets you countered.

## Pad / stick

- Use the **default stick deadzone** unless drift forces a change; raising it
  dulls your first step and dribble inputs.
- Many top players use **back-paddle controllers** (DualSense Edge / Scuf) mapped
  so they can sprint (R2) and flick the R-stick **without taking the thumb off
  movement**. Common maps: paddle = ○ (tackle) and paddle = R1 (player run /
  L1 second-man press). Not required, but it removes claw-grip and improves
  defending and skill-move access.

## Pre-match checklist

1. Confirm gameplay mode (Competitive online).
2. Set your **tactics + custom plan** (see `02`) — don't trust EA defaults.
3. Check **stadium/pitch** (longer/wetter pitch = faster ball = better for
   possession passing).
4. Set **set-piece takers** (best Power Shot / Dead Ball / Set Piece Specialist).
