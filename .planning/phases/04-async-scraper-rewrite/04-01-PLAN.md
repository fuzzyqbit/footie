- Module constants: `USER_AGENT = "footie-playbook/0.1 (personal squad tool)"` (identical to web.py:9); `_LIMITS = httpx.Limits(max_connections=8, max_keepalive_connections=8, keepalive_expiry=30.0)`; `_TIMEOUT = httpx.Timeout(connect=10.0, read=15.0, write=10.0, pool=5.0)`.
- `class HostRateLimiter`: `__init__(self, min_interval: float)`; holds `self._min`, `self._next: dict[str, float] = defaultdict(float)`, `self._locks: dict[str, asyncio.Lock] = defaultdict(asyncio.Lock)`. `async def wait(self, host: str)`: `async with self._locks[host]:` compute `delay = self._next[host] - time.monotonic()`, `if delay > 0: await asyncio.sleep(delay)`, then set `self._next[host] = time.monotonic() + self._min + random.uniform(0, self._min)` (jitter preserves the old jittered_sleep anti-metronome property). The per-host lock serializes the gate so two coroutines to the same host can't both pass instantly.
- `class AsyncFetcher`: `__init__(self, *, concurrency: int = 4, min_interval: float = 1.0, retries: int = 1)` → `self._sem = asyncio.Semaphore(concurrency)`, `self._rl = HostRateLimiter(min_interval)`, `self._retries = retries`, `self._client: httpx.AsyncClient | None = None`. `async def __aenter__`: create `httpx.AsyncClient(limits=_LIMITS, timeout=_TIMEOUT, follow_redirects=True, headers={"User-Agent": USER_AGENT})`; return self. `async def __aexit__(self, *exc)`: `await self._client.aclose()`.
- `async def fetch(self, url: str) -> str`: `host = urlsplit(url).netloc`; loop `for _ in range(self._retries + 1):` → `await self._rl.wait(host)` (politeness gate, OUTSIDE the semaphore so the wait doesn't hold a slot) → `async with self._sem:` try `resp = await self._client.get(url); resp.raise_for_status(); return resp.text` except `httpx.HTTPError as exc: last = exc` then `await asyncio.sleep(random.uniform(0.0, 0.5))` (modest backoff, only on the retry path, strictly politer). After the loop: `raise FetchError(f"could not fetch {url}: {last}")` — wording must match web.py:26 exactly (variable name aside, the f-string text is identical).

Imports: `from __future__ import annotations`; `import asyncio, random, time`; `import httpx`; `from collections import defaultdict`; `from urllib.parse import urlsplit`; `from ..errors import FetchError`.

Do NOT use `AsyncHTTPTransport(retries=)` — it only covers ConnectError/ConnectTimeout; we need 1 retry on ANY HTTPError to match web.py. Add a short module docstring + inline comments mirroring the research notes (why both semaphore and rate limiter; why manual retry).
